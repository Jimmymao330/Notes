染色體

科學史

1879年 弗萊明發現細胞染色後可以看見絲狀物
1888年 科學界正式將此絲狀物命名為「染色體」

組成

  • 由染色質聚合而成

構造

  • 交叉處稱為中結
  • 中結上有著絲點，由兩個蛋白質所組成

功能

  • 只有在要複製後要準備分裂時才會形成二分體（常見的X形染色體）
  • 分裂前不會看到染色體
Important

並非「染色體複製」而是「染色質複製後纏繞成染色體」

分類

同源染色體
  • 由兩個親代分別提供一半
  • 型態與構造基本相同
Note
  • 人類是雙倍體，一對X一組O同源染色體有兩條（2n）。
  • 小麥是七倍體，一組同源染色體有七條（7n）。
  • n 是單位，代表一組同源染色體中有幾條。
姊妹染色體
  • 染色體複製後的兩條相同染色體，即為二分體中的兩股。
  • 透過中結來計算，兩條共用一個中結

減數分裂

1. 前置作業（間期）

完成染色質與胞器的複製

2. 細胞核分裂

減數分裂第一階段
  1. 染色質纏繞成姐妹染色體，以中結連接成二分體
  2. 聯會[1]，形成四分體
  3. 中心粒延伸出紡錘絲，並與二分體著絲點連接
  4. 中心粒拉著四分體移動至細胞中央排成一排
  5. 同源染色體分離，兩個二分體分別分離到不同的子細胞中，並向細胞兩堆移動
減數分裂第二階段
  1. 中心粒延伸出紡錘絲，並與二分體著絲點連接（染色體不複製）
  2. 中心粒拉著二分體移動至細胞中央排成一排
  3. 姐妹染色體分離

3. 細胞質分裂

  1. 形成分裂溝
  2. 分配胞器
  3. 子細胞形成（例如形成人體配子）

  1. 聯會：指兩個二分體（同源染色體）相疊，形成四分體。↩︎

減數分裂

有絲分裂

1. 前置作業（間期）

完成染色質與胞器的複製

2. 細胞核分裂

  1. 染色質纏繞成姐妹染色體，以中結連接成二分體
  2. 中心粒附近形成出紡錘絲，並與二分體著絲點連接
  3. 中心粒拉著二分體移動至細胞中央排成一排
  4. 中心粒協助姐妹染色體分離，並向細胞兩堆移動

3. 細胞質分裂

動物細胞（沒有細胞壁）
  1. 形成分裂溝
  2. 分配胞器
  3. 子細胞形成
植物細胞（有細胞壁）
  1. 用液泡集中來將纖維素排列到中間
  2. 液泡融合來讓纖維素相織再一起，形成細胞板
  3. 分配胞器
  4. 子細胞形成

原核細胞與真核細胞

原核細胞

  • 沒有細胞核 → 也沒有膜狀胞器
  • 大多數具有細胞壁（例外：黴漿菌）
  • 雖然沒有細胞核，但有遺傳物質。遺傳物質會在細胞質中的特定區域，沒有被膜包起來。
  • 有核糖體

真核細胞

  • 真的有細胞核的細胞
  • 細胞內有細胞核 → 有膜狀胞器
  • 通常會有原核細胞的約10倍大
  • 通常約為 10~100 𝜇𝑚

能量載體

Note

ATP 和 ADP 都是核苷酸
組成：

  • "A"是指含氮鹼基（ATGCU）中的腺嘌呤（A）
  • 組成 ATP/ADP 的另一要素是核糖
  • ATP/ADP 還有包含一個磷酸基

ATP 腺苷三磷酸

  • 有三個磷酸基
  • 在 ATP 變成 ADP 的過程中，會有一個磷酸基脫離，並釋放能量。

ADP 腺苷二磷酸

  • 有二個磷酸基
  • 在 ADP 變成 ATP 的過程中，會加上一個磷酸基，並吸收能量。
Note

計算生物體能量含量時，通常使用ATP與ADP的比值，而非ATP的絕對含量。

能量載體

代謝作用

同化代謝

  • 小分子合成成大分子
  • 反應所消耗的能量 > 釋放出的能量
Example
  • 葡萄糖合成為肝醣

異化代謝

  • 大分子分解成小分子
  • 釋放出的能量 > 反應所消耗的能量
Example
  • 葡萄糖分解為二氧化碳與水
